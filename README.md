# Stonks Everyday

Android application targeting Android 16 (API 36).

## 專案資訊

- **Target SDK**: Android 16 (API 36)
- **Min SDK**: Android 7.0 (API 24)
- **語言**: Kotlin
- **建構工具**: Gradle 8.9
- **Android Gradle Plugin**: 8.7.3

## 建構專案

使用 Gradle Wrapper 建構專案:

```bash
./gradlew build
```

## 安裝到裝置

```bash
./gradlew installDebug
```
